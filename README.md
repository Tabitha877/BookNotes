**BookNotes** is a website where you can keep track of the books you have read.
***

## Features
* Work with Express and Node js
* Book info & covers from [OpenLibrary.org](https://openlibrary.org/)
* CRUD method (Create Read Update Delete)
  * Add book
  * Get book info
  * Edit book
  * Delete book
* PostgreSQL Database to store the book info
* Search option to search books on title
* Sort option to sort books by:
  * Name 
  * Rating
  * Date
* Error handling

## Dependencies
* pg
* ejs
* express
* body-parser
* axios
